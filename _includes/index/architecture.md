# Quickstart

```sh
$ git clone https://github.com/fabiandev/vue-twitter-stream-app.git
$ cd vue-twitter-stream-app
$ cp server/.env.example server/.env
$ yarn
```

Now fill in your credentials in `server/.env` and run `yarn start`.
